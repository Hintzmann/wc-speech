How to use

Click 🗣️ Speech tool in the top-right corner to open the toolbar. Press ▶️ play to have this page spoken. Use the ⏪ previous and ⏩ next buttons to jump between sentences, and ⏸️ pause to stop at any time.

Open ⚙️ Voice options to choose a voice, change the speed, or turn on scroll-to-follow so the page keeps the current sentence in view.

Markup behaviour

Compact English samples — each block shows one kind of markup. Use prev/next sentence to step through and hear what the component does with each element.

Images

Informative images are spoken from their alt text. The portrait sits beside the text. Decorative images with an empty alt are skipped — the next image has no spoken announcement.

Decorative image:

Abbreviations

An abbr with a title is spoken as the expansion: EAA.

Dates

A time element can add a localized date from datetime: the exhibition opened on 23 June 2014.

Code

Inline and block code is read as its text content in code-lang (default English): 

console.log('Hello, world!');

Links

Link text stays in the sentence flow — you should hear “full article” inline: read the full article for the complete speech.

Edits

Insertions and deletions are read as plain text: price was 100 kr. 75 kr. after the discount.
