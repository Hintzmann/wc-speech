Simple integration

This page uses one button wired to command="--playpause". There is no speech bar, voice picker, or speed control — defaults only.

The button and <wc-speech> sit outside the reading target (#article). Point target at the content to speak; keep controls outside when you can so they are not collected as readable text.

If <wc-speech> is inside the target, the component still skips its own subtree when collecting text.

Press 🗣️ Læs op to hear this article. Use the same button to pause.
